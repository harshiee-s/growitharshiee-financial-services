Each page is rendered to a canvas, re-encoded as a JPEG at your chosen quality, and embedded into a fresh PDF. This typically cuts file size by 50–80%. Note: the output becomes image-based — text is no longer selectable. Use "Low" compression if you need to preserve selectable text.
Encrypted PDFs need to be unlocked first. Open them in your PDF viewer with the password, then re-save as an unprotected copy — the toolkit will accept that.
