This tool reads your 26AS PDF directly in your browser (nothing is uploaded to any server) and produces a clean Excel workbook with every deductor auto-separated and every transaction itemised — ready for reconciliation, audit, or ITR filing.

🔍 What This Tool Extracts

The parser auto-detects the TRACES 26AS layout and extracts every data point needed for reconciliation:

📌 Assessee Metadata

  • PAN, Status, and Assessee Name
  • Financial Year & Assessment Year
  • File name and extraction timestamp

🏢 Deductor Summary (Sheet 1)

  • Serial number from the original PDF
  • Full Deductor Name & TAN
  • Total Amount Paid / Credited
  • Total Tax Deducted & Total TDS Deposited

📊 Detailed Transactions (Sheet 2)

  • Every transaction line with its parent deductor mapped
  • Section code (194C, 194JA, 194JB, 194H, 194A, etc.)
  • Transaction Date, Status of Booking, Date of Booking
  • Remarks (lower deduction certificate, rectifications, reversals)
  • Amount Paid, Tax Deducted, TDS Deposited — as numeric values
